Twin AI is a scalable Virtual Try-On AI application that enables users to visualize outfits digitally using AI-powered image processing.

I contributed as a Mobile Engineer, building a production-ready prototype with a focus on performance, scalability, and heavy image workflow optimization.

🔥 Key Contributions

Integrated AI APIs for real-time virtual try-on processing

Designed scalable backend architecture for high image-load workflows

Implemented background processing & caching to optimize performance

Built Flutter-based mobile application with clean modular architecture

Developed REST APIs & backend services for seamless AI integration

🛠 Tech Stack

Flutter • Dart • FastAPI • Python • REST APIs • LLM APIs • WebSockets • Firebase • AWS/GCP

📈 Impact

Delivered a production-ready prototype

Engineered technical scalability contributing to ₹40Cr valuation (Shark Tank S4 mention)

Optimized heavy image workflows for real-world AI deployment
